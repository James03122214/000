# 🚨 台股處置股預警系統

## 📋 專案簡介

這是一個基於 Streamlit 的台股處置股預警系統，能夠即時抓取台股真實市場數據，分析股票是否符合處置股標準，並提供視覺化風險預警介面。

### ✨ 主要功能

- **即時數據抓取**: 串接 Yahoo Finance API 獲取真實市場數據
- **智能分析**: 基於台灣證券交易所標準進行處置股分析
- **風險預警**: 多層次風險等級判定（正常、注意股、潛在處置股、高風險處置股）
- **股票名稱顯示**: 自動顯示股票中文名稱
- **視覺化介面**: 直觀的價格走勢圖和風險指標
- **多市場支援**: 支援台灣上市 (.TW) 和上櫃 (.TWO) 股票

### 🎯 適用場景

- 個人投資者風險管理
- 股票分析工具
- 教學示範
- 投資研究

## 📊 處置股判斷標準

本系統基於台灣證券交易所標準進行分析：

- **6日漲跌幅**: ±25%
- **10日漲跌幅**: ±50%
- **30日漲跌幅**: ±100%
- **成交量異常**: 成交量異常放大
- **週轉率過高**: 週轉率超過正常範圍
- **連續3日達標**: 連續3日達到注意股標準

## 📱 使用說明

1. **輸入股票代號**: 在左側邊欄輸入股票代號（如：2330）
2. **點擊分析**: 點擊「開始分析」按鈕
3. **查看結果**: 
   - 股票基本資訊（名稱、市場、行業）
   - 今日收盤數據（價格、成交量）
   - 風險預警判定
   - 近期價格走勢圖

### 快速測試

系統提供快速測試按鈕：
- **測試 2330**: 測試台積電（正常股票）
- **測試高風險股**: 測試高波動性股票

## 📈 支援的股票

系統支援所有台灣上市和上櫃股票，包括：

- 大型權值股（台積電、聯發科、鴻海等）
- 半導體相關股票
- 電子股
- 金融股
- 傳產股
- ETF（如：元大台灣50）
